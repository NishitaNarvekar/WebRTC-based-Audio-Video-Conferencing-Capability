There has been a huge development in Web technologies with new advancements like standardisation of protocols - HTML and HTTP. But this growth is not yet seen in areas which have service dependencies, like audio and video. Even now, video on internet requires help of applications like plugins. However, with emergence of HTML5 and WebRTC, it is now possible to enable audio and video communication using standard browser. HTML5 provides tags which allow video to be seen directly on browser. WebRTC technology, an intiative by Google, provides some APIs that can easily access local and remote media stream.
